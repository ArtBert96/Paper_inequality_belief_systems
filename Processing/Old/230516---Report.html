Regressions

We have different ways to do the final regression (cca variable + predictors -> att_redis).

Design A: Based on EGA communities

First solution: We fit EGA on the full samples US and NL; we create means indexes starting from EGA groups; we do this because we rely on this: we can state the following relationship as a fundamental rule of network psychometrics: Clusters in network = latent variables. Paper link. However, here we have a problem, since these indexes have poor Cronbach alpha, and if we try to fit this structure with a CFA we have poor fit too. Therefore if we go on with this route we can say we are estimating mean indexes of each community, but it is hard to state we are building mean indexes of latent factor. Still, if we take this route we have the best regression results (CCA variable significant, increase in R2, only for NL)

Later, we can put all these indexes and variables in the regression. Results say we have a significant effect of the CCA variable and an increase in the R2, but only for NL. Additionally, Cronbach alphas still present super low values in some indexes.
